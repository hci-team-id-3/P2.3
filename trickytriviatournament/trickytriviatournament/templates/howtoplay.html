Welcome to the Tricky Trivia Tournament!

Background Information:

This trivia game has a twist compared to your typical trivia game. The game will tests the players knowledge in the following 5 categories

  1. Texas Tech University
  2. Computer Science
  3. Food
  4. Sports
  5. Anime/Manga
The game will auto-generate 2 questions from each of these categories to make up the game round.

How to Play:

Before the game starts, a counter will count down prompting the user to get ready. The questions will appear one by one with the question at the top of the screen and the four (4) answer choices listed below.

The player should pick their correct answer at which point the game will let the player know whether they got it right or not. If the answer was right, then the answer button will highlight green and a ding sound effect will alert the player before moving on to the next question.

If the answer was incorrect, the answer button will highlight red and a buzzer will sound to alert the player before moving on to the next question.
